

<h1>시작하기</h1>
<a name="section_1_1"></a>
<h2>1. 앱을 등록합니다.</h2>

* 새로운 앱을 만듭니다. <a href="https://gamekiki.com/sub/sub01_application.php" name="create_app_link" target="_blank">앱 만들기</a>

* 생성된 앱의 Application id 와 Application secret 을 메모합니다.
![](http://gamebin.iptime.org/gamekiki/metaadmin/assets/data/manual/section_1_1.png)

* 미니게임 사용을 원하시는 경우에는 <a name="create_mini_game_link" href="#">미니게임 등록</a> 페이지를 참조하시기 바랍니다. 


<a name="section_1_2"></a>
<h2>2. 스크립트를 삽입합니다.</h2> 

위젯을 사용할 페이지에 키키 스크립트를 넣습니다.

```javascript
<html>
	<head>
		<meta charset="utf-8" />
		<script src="http://gamebin.iptime.org/kshop/kwidget/kwidget.js"></script>
		</head>
</html>
```
<a name="section_1_3"></a>
<h2 id="english">3. 로그인이 되어있지 않을 경우 이동될 url을 설정합니다.</h2>

로그인 유무를 체크해서 로그인이 되어있지 않을 경우 이동될 로그인 url을 설정합니다.
```javascript
<html>
<head>
	<meta charset="utf-8" />
	<script src="http://gamekiki.com/kshop/kwidget/kwidget.js"></script>
	<script>
		kshopMds.loginPage = "./login.html"; 
	</script>
</head>
</html>
```
<a name="section_1_4"></a>
<h2>4. 샘플 앱의 init 코드를  설정합니다 </h2>
* [생성된 앱](#create_app_link)의 appId, 사용자의 userId 와 함께 init 코드를 설정합니다.
* [미니게임](#create_mini_game_link)을 등록해서 사용을 원하실 경우에는 mgId를 파라미터로 주시면 됩니다.

```javascript
<html>
<head>
	<meta charset="utf-8" />
	<script src="http://gamekiki.com/kshop/kwidget/kwidget.js"></script>
	<script>
	
		var appId="app_L6l71RIp00000";
		var mgId = "MGMAI_G5d0u045";
		
		kshopMds.loginPage = "./login.html"; 
		kshopMds.init({
		 appId : appId,
		 mgId : mgId,
		 userId : "test01"
		});		
	</script>
</head>
</html>

```
<a name="section_1_5"></a>
<h2>5. 사이트 사용자의 쿠키 만료기간을 체크합니다.</h2>

사이트 사용자의 쿠키 세션이 만료된 경우, 키키위젯의 세션 또한
만료시켜야 합니다. 

init 코드에 autoLogin 값을 'N'으로 넣으시면 키키위젯의 세션이 만료됩니다.

```php
<html>
<head>
	<meta charset="utf-8" />
	<script src="http://gamekiki.com/kshop/kwidget/kwidget.js"></script>
	<script>
		// 서버사이드 사용자 쿠키 정보
		var user_session_cookie = "<%=Response.cookies("user_login_session")%>";
		var autoLogin= (user_session_cookie)? "Y" : "N";
		
		var appId="app_L6l71RIp00000";
		var mgId = "MGMAI_G5d0u045";
		
		kshopMds.loginPage = "./login.html"; 
		kshopMds.init({
		 appId : appId,
		 mgId : mgId,
		 userId : "shleetest01",
		 autoLogin : autoLogin
		});		

	</script>
</head>
</html>
```
