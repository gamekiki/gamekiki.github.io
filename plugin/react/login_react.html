    로그인 연동


    1. 시작하기 에서 했던 KikiDiv 세팅을 로그인 페이지에도 똑같이 해줍니다.




    2. kikiCallback 함수에서 KikiDiv의 로그인 함수를 호출합니다.

    Login Response

    Name Type Description
    okFlag boolean 로그인 성공여부
    true=로그인 성공
    false= 로그인 실패
    res object 로그인 호출이후 키키서버로부터 받은 response 값

    3. 로그인 성공여부를 체크해서 성공 시 메인페이지로 이동합니다.
